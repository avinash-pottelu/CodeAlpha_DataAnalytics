# Task 1: Exploratory Data Analysis (EDA)

This project is part of the CodeAlpha Data Analytics Internship.

In this task, Exploratory Data Analysis was performed on a student performance dataset
to understand academic trends and patterns.

## Dataset
Student Performance Dataset containing information such as gender, lunch type,
test preparation course, and subject scores.

## Analysis Performed
- Checked for missing and duplicate values
- Calculated average math, reading, and writing scores
- Performed gender-wise, lunch-wise, and test preparation-wise analysis
- Created pivot tables and visualizations

## Tools Used
- Microsoft Excel

## Key Insights
- Female students performed better in reading and writing.
- Male students scored slightly higher in mathematics.
- Students who completed test preparation performed better overall.
